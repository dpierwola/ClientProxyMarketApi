## Instalation

### Git
Clone repository using git clone.

```bash
$ git clone https://github.com/dpierwola/ProxyMarketApi.git
```

Go to project dir

```bash
$ cd ProxyMarketApi
```

Install dependencies
```bash
$ composer install
```
